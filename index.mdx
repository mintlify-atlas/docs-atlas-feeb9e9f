## What is MVC PHP?

MVC PHP is a prototype framework built to provide a structured MVC architecture for PHP applications. It ships with everything you need to build modern web applications — from a clean routing layer and an Eloquent-inspired ORM to session authentication, API tokens, a WebSocket server, and a CLI scaffolding tool.

## Key features

<CardGroup cols={2}>
  <Card title="MVC Architecture" icon="layer-group">
    Clean separation of concerns with controllers, models, and blade-style views.
  </Card>
  <Card title="Eloquent-inspired ORM" icon="table">
    HasOne, HasMany, and ManyToMany relationships with a fluent query builder.
  </Card>
  <Card title="Migrations & Seeders" icon="arrows-rotate">
    Version-controlled database schema with up/down migrations and data seeders.
  </Card>
  <Card title="Authentication" icon="lock">
    Session-based login and API Bearer token auth with optional PAM support.
  </Card>
  <Card title="Middleware Pipeline" icon="filter">
    Attach middleware to individual routes or groups for request filtering.
  </Card>
  <Card title="CLI Manager" icon="terminal">
    Scaffold controllers, models, migrations, seeders, middleware, and more.
  </Card>
  <Card title="HTTP Client" icon="globe">
    Built-in Guzzle-backed client for outbound HTTP requests.
  </Card>
  <Card title="WebSocket Server" icon="plug">
    Simple Socket.io-inspired server for real-time features.
  </Card>
</CardGroup>
